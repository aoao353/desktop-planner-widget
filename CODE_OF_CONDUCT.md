# 社区行为准则

## 我们的承诺

为了营造开放、友好的环境，我们承诺：参与者的年龄、体型、身体健全与否、民族、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向均不应成为遭受骚扰或不当对待的理由。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友善、包容的语言  
- 尊重不同的观点与经验  
- 建设性地接受批评  
- 关注对社区最有利的事情  
- 对其他社区成员表示同理心  

不可接受的行为包括：

- 使用性化的语言或图像，以及不受欢迎的性关注或示好  
- 挑衅、侮辱/贬损的评论，以及人身或政治攻击  
- 公开或私下骚扰  
- 未经明确许可发布他人的私人信息（如物理或电子地址）  
- 其他在专业场合中合理认为不恰当的行为  

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并会对被认为不当、威胁、冒犯或有害的行为采取适当、公平的纠正措施。

维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、issue 等，并会在必要时暂时或永久禁止任何参与者的贡献行为。

## 适用范围

本行为准则适用于所有项目空间，以及个人在公共场代表本项目或其社区时的行为。代表项目包括使用官方项目邮箱、通过社交媒体账号发帖、或在在线或线下活动中担任指定代表等场景。

## 执行

可通过 [SECURITY.md](./SECURITY.md) 中的联系方式报告辱骂、骚扰或其他不可接受的行为。维护团队将审查并调查所有投诉，并以认为适当的方式回应。维护团队有义务对报告事件者保密。具体执行政策的更多细节可另行发布。

未真诚遵守或执行本行为准则的项目维护者，可能被项目其他领导成员决定永久撤销其贡献。
